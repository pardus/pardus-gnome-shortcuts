# Pardus Gnome Shortcuts

Pardus Gnome Shortcuts is a user-friendly application designed to display the default and custom keyboard shortcuts available in the Pardus GNOME desktop environment. 

It provides an intuitive interface where users can browse through various categories of shortcuts such as System, Launchers, Screenshot Tools, and Workspace management. Users can toggle between a modern grid view and a detailed list view to find the information they need efficiently.

### Tech Stack

This project is built using the following technologies:

- **Language:** Python 3
- **GUI Framework:** GTK 4
- **Library:** Libadwaita (Adwaita)
- **Build System:** Meson
